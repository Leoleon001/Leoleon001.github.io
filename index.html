Prueba de Deployment en GitHub

¡Bienvenido a mi página de prueba!

Esta es una página HTML simple creada para probar el proceso de deployment en GitHub Pages.

Instrucciones para el deployment:

  1. Crea un repositorio en GitHub
  2. Sube este archivo HTML (nómbralo index.html)
  3. Ve a Settings > Pages
  4. Selecciona la rama principal (main o master)
  5. Haz clic en Save

Si todo funciona correctamente, esta página debería estar disponible en tu dominio de GitHub Pages en unos minutos.

Puedes modificar este código para añadir más contenido y probar diferentes características.
